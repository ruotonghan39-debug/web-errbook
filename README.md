# 错题本闭环 Demo（Web 原型）

纯前端（HTML/CSS/JS）原型，用浏览器 `localStorage` 模拟数据落库，演示端到端闭环：

`错题录入 → 老师讲解 → 学生复习（隐藏答案）→ 掌握判定 → 智能再练 → 导出/打印`

## 使用

打开：`web-errbook/index.html`

## 功能（MVP）

- 录入：手动填写字段（可选图片，Base64 本地存储）
- 老师：添加讲解文字与标准步骤，并推动进入复习
- 学生：复习隐藏答案；提交后系统比对（简化为文本归一化对比）+ 信心评分（1-5）
- 再练：从静态题库中推荐并允许提交完成；完成后归档
- 导出：打印版错题本（浏览器打印/PDF）与 JSON 导出
